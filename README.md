# Personal website

Personal portfolio site for [muellner.dev](https://muellner.dev), built with Astro, Tailwind CSS, and DaisyUI. Content is managed via YAML files in `src/data/`.

## Prerequisites

- Node.js v23+

## Usage

```bash
# Install dependencies
npm install

# Start dev server (http://localhost:4321)
npm run dev

# Build for production
npm run build

# Preview production build locally
npm run preview
```

## Deployment

Pushes to `main` automatically deploy to GitHub Pages via the workflow in `.github/workflows/gh-pages.yaml`.
